                                                 -: Save & Delete Search in Loggly :-

Follow the below steps to save a search in Loggly -

-> Navigate to dashboard.
-> Select type of logs under filiter.
-> select type of service.
-> Clikc on faviourite option.
-> Select "Save this search as..." in drop down.
-> Give name bases on you search to identify 
-> Click on save 


* Follow the below steps to remove a saved search in Loggly -

-> Navigate to the Search page (where logs are displayed).
-> Click on the Saved Search dropdown (⭐ icon or saved search name).
-> Select “Manage Saved Searches”.
-> Locate the saved search you want to delete.
-> Click on the Delete (🗑 / X) icon next to it.
-> Confirm the deletion when prompted.


* To delete an alert on loggly follow the below steps -
-> Go to the Alerts section from the top menu.
-> Click on Alert List.
-> Find the alert you want to delete or modify.
-> Click on the Delete (🗑 / X) icon next to the alert.
-> Confirm the deletion.
