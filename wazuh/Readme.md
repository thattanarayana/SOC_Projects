                                  -:  Wazuh Open-Source SIEM Implementation and Security Monitoring Lab Project  :-                                            

Project Description -

This project was developed as part of my cybersecurity learning journey to gain hands-on experience with Security Information and Event Management (SIEM) technologies and Security Operations Center (SOC) workflows.

In this project, I deployed Wazuh SIEM in both On-Premises and Cloud environments, integrated Windows and Linux endpoints, and configured centralized log collection and security monitoring. The objective was to understand how security events are generated, collected, correlated, and transformed into actionable security alerts.

The lab simulates real-world SOC operations, including alert monitoring, incident triage, log analysis, threat intelligence, detection engineering, and incident investigation. Multiple attack scenarios were performed to validate detection capabilities and improve understanding of how security teams identify and respond to threats.

Key Activities Performed - 

* Deployed Wazuh SIEM using Docker Desktop (On-Premises)
* Configured and monitored Wazuh Cloud Platform
* Installed and managed Wazuh Agents on endpoints
* Collected and analyzed Windows and Linux security logs
* Created and tested custom detection rules
* Performed alert validation and incident investigation
* Conducted IOC analysis using threat intelligence sources
* Classified incidents as True Positive or False Positive
* Documented investigation findings and response actions

Attack Simulations -
-> Linux SSH Brute Force Attack Detection
-> Unauthorized File Access Detection
-> Windows Privilege Escalation Detection
-> Port Scanning Detection
-> Linux Suspicious Command Activity Detection

Skills Demonstrated - 
* SIEM Administration
* Security Monitoring
* Log Analysis
* Event Correlation
* Incident Triage
* Incident Investigation
* Threat Hunting
* Detection Engineering
* Threat Intelligence Analysis
* SOC Operations

Project Outcome -
Successfully built a functional SOC lab environment using Wazuh SIEM and gained practical experience in security monitoring, alert investigation, threat detection, and incident response workflows. This project helped strengthen my understanding of real-world SOC operations and prepared me for entry-level cybersecurity roles such as SOC Analyst, Cybersecurity Analyst, and Security Monitoring Analyst.
